# Simvo Vibe Coder


This project demonstrates:

- API Requests
- JSON Data Extraction
- Control Flow
- Primitive Data Types
- Modular Python Project Structure

This project fetches species data from the GBIF API and extracts:

- Scientific Name
- Authorship
- Kingdom
- Habitats
- Threat Status
- Vernacular Name (English priority)

## Run Project

1. Install dependencies:
pip install -r requirements.txt

2. Run:
python main.py
